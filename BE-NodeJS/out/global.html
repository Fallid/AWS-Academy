Methods

requestListener(request, response)

Fungsi `requestListener` adalah handler utama untuk setiap permintaan HTTP yang masuk ke server. Fungsi ini menangani berbagai URL dan metode HTTP, mengembalikan konten HTML yang berbeda atau pesan kesalahan berdasarkan permintaan.
Parameters:
Name Type Description
request IncomingMessage Objek permintaan HTTP yang berisi informasi terkait permintaan klien, seperti metode HTTP dan URL.
response ServerResponse Objek respons HTTP yang digunakan untuk menanggapi permintaan, termasuk mengatur header, kode status, dan mengirimkan isi respons kembali ke klien.
